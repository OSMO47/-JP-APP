# JLPT Vocabulary Trainer

เว็บแอปสำหรับฝึกคำศัพท์ภาษาญี่ปุ่นรองรับ JLPT ระดับ N5, N4 และ N3 พัฒนาด้วย Next.js และ Tailwind CSS

## คุณสมบัติหลัก

- หน้าหลักเลือกโหมดและระดับ JLPT ที่ต้องการฝึก
- โหมดสุ่มคำศัพท์พร้อมช้อยส์ 4 ตัวเลือก และสถิติการทำแบบทดสอบ
- โหมดคลังคำศัพท์ของฉัน สำหรับบันทึกคำศัพท์เพิ่มเติมลงใน Local Storage
- รองรับการใช้งานทั้งภาษาไทยและภาษาญี่ปุ่น พร้อมดีไซน์ที่ตอบสนองทุกอุปกรณ์

## การเริ่มต้นพัฒนา

1. ติดตั้งแพ็กเกจที่จำเป็น

   ```bash
   npm install
   ```

2. เริ่มโหมดพัฒนา

   ```bash
   npm run dev
   ```

3. เปิดเบราว์เซอร์ที่ `http://localhost:3000`

## โครงสร้างโฟลเดอร์สำคัญ

```
/pages
  index.js        # หน้าเลือกระดับและโหมด
  quiz.js         # โหมดสุ่มคำศัพท์
  my-vocab.js     # คลังคำศัพท์ของฉัน
/components
  WordCard.js     # การ์ดแสดงคำศัพท์
  QuizOption.js   # ปุ่มตัวเลือกคำตอบ
/data
  vocab_n5.json
  vocab_n4.json
  vocab_n3.json
```

## การดีพลอย

รันคำสั่ง build และ start สำหรับสภาพแวดล้อม production

```bash
npm run build
npm run start
```
